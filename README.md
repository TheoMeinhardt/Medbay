## start Express-Server

1. `npm i`
2. `npm run build`
3. `npm start`

## IDE-config

1. `npm i --inlcude=dev`
2. ` npm run dev`
3. `cd src`
4. `tsc -p ./tsconfig.data.json`
5. `cd src`
6. `tsc -p ./tsconfig.public.json`
7. `cd public/js`
8. `watchify script.js -o bundle.js`

extension live-sass-compiler installieren und den `watch sass` - button clicken
